Trabalhando com áudios.

Patra trabalharmos com áudios usamos a tag <áudio>.

Inserindo áudio

A tag áudio tem importantes atributos como preload que indica que o áudio será pre-carregado ou não, este atributo possui 3 valores:

  • metadate, carrega apenas as informações sobre tamanho, tempo =, direitos autorais, etc., do arquivo.
  • none, não carrega nada até que o play seja ativado.
  • auto, carrega o áudio assim que a página é carregada, caso nehuma dessas alternatvas esteja ativa, ele está ativo por padão.

Consols é o atributo que mostra os controles do player, caso não esteja ativo o audio não será controlado pois os controles não estarão visíveis.

O atributo autoplay reproduz o áudio aassim que a página é carregada.

Loop reproduz o áudio infinitamente quando o tempo termina.

No código, coloque o áudio que deseja executar primeiro, depois coloque outras versões para o caso de o navegador não ser compatível, por fim, você também pode adicionar um parágrafo disponibilizando o áudio para download caso nenhuma das alternativas funcione.

    
        <audio>
            <source src="minia/audio.mp3" type="audio/mpeg">
            <source src="minia/audio.ogg" type="audio/ogg">
            <source src="midia/audio.wav" type="image/wav">
            <p>Seu navegador não suporta audio <a href="midia/audio.mp3"
             douwnload="audio.mp3" type="audio/mpeg">&lr;/a></p>
        </audio>

Assim como as imagens, os áudios precisam de versões em outros formatos par oferecermos uma melhor experiência aos usuários. Nem todos os navegadores aceitam todos os formatos.

Formatos de áudios

Os formatos de áudios mais usados são:

    • .MP3
    • .OGG
    • .WAV
