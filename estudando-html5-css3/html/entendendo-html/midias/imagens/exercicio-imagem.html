Trabalhando com imagens em HTML 5

Para inserir imagens em um site é preciso usar a tag <img>.

Inserindo imagens

imagem da mesma pasta:

  • <img src="p.png" alt="Descriçção">

    • Digite "img" no VSCode + enter. Em "scr" localize a imagem na mesma pasta usando as teclas "Ctrl + espaço", em "alt" adicione uma breve descrição sobre a imagem, essa descrição ajuda deficientes visuais a "verem" sua página. Além disso, os mecanismos de busca conseguem mostrar a sua imagem nas pesquisar mais facilmente.

"Src" vem de "Source" que significa "Origem".

Inserindo imagens de outra pasta

  • <img src="Imagens/m.png" alt="Descrição">

    • Nesse caso a imagem está na pasta "imagens". Para acessar a pasta digite "../" se a imagem estiver na pasta anterior ou "../../" se estiver em duas pastas anteriores.

Inserindo imagem externa (da internet)

  • <img src="https://png.pngtree.com/png-clipart/20190630/original/pngtree-html-file-document-icon-png-image_4174779.jpg" alt="Descriçção">

    • A única diferença aqui é que você vai precisar de um link com a imagem disponível e colocar em "src". O problema com essa opção é que se a imagem for removida do local ou ficar privada, não poderá ser acessada e sua página perderá a imagem.

Tipos e tamanhos de imagens

Os formatos mais aceitos pelos navegadores são:

  • PNG

    • PNG é um formato de dados utilizado para imagens, que surgiu em 1996 como substituto para o formato GIF, devido ao facto de este último incluir algoritmos patenteados. (fonte: Wikipédia).

  • JPG

    • JPG é um método comum usado para comprimir imagens fotográficas. O grau de redução pode ser ajustado, o que permite a você escolher o tamanho de armazenamento e seu nível de compromisso com a qualidade da imagem. Geralmente se obtém uma compressão pouco perceptível na perda de qualidade da imagem. (Fonte: Wikipédia).

  • GIF

    • Graphics Interchange Format ou GIF é um formato de imagem de bitmap que foi desenvolvido por uma equipe do provedor de serviços on-line CompuServe, liderado pelo cientista de computação americano Steve Wilhite em 15 de junho de 1987. (fonte: Wikipédia).

Como encontrar imagens?

Esses são alguns sites onde podemos encontrar imagens, algumas gratuitas e com autorização de uso:

  • UnSplash
  • Pexels
  • FreePik
  • Rawpixel
  • Pixabay
  • Libre Shot
  • Wikimedia Commons

É sempre importante ler bem as regras de cada site e ter certeza de que são mesmo gratuitas ou com direito de uso livre.

Outra coisa! Verifique o tipo e o tamanho das imagens que vai precisar e gere exatamente o que precisa, imagens muito grandes e pesadas tornam o site difícil de carregar e os buscadores acabam deixando eles de lado.
Alinhe seu site de acordo com as exigências dos maiores buscadores, isso te fará ganhar competitividade em relação a outros desenvolvedores.

Site com imagens responsivas

Para tornar um site com imagens responsivas vamos usar as tags "picture", "source" e "img".
A tag <picture> é um container usado para especificar múltiplos elementos, dentro desse container inserimos a tag <source> onde vamos configurar o tamanho da imagem com o atributo media="(max-width: 1050px)", indicar a imagem que será carregada pelo navegador usando srcset="imagem.png", para quando atingir o tamanho máximo da width e type="imagem/png" indicando o tipo do arquivo que será carregado. Veja mais sobre os media types aqui.

Exercício imagem responsiva

O objetivo desse exercício é abrir o site em diversos formatos, para isso a imagem precisa se adaptar a diferentes tamanhos.

Nesse caso vamos usar as tags com seus atributos e organizar as tags "source" do menor para o maior.

                    
        <picture>
            <source media="(max-width:750px )" srcset="imagens/p.png" type="image/png">
            <source media="(max-width: 1050px)" srcset="imagens/m.png" type="image/png">
            <img src="imagens/g.png" alt="Imagem flexivel">
        </picture>

Site com imagem de fundo

Para adicionarmos uma imagem no fundo do site precisamos usar adicionar na tag style ou a folha de CSS o atributo background-image.

            
<style>
    body{
        background-image: url('Imagens/g.png');
    }
</style>

Dessa mesma forma podemos usar url externa, basta adicionar a url desejada.

Personalizando o background

Para personalizarmos o fundo, caso a imagem seja pequena, utilizamos o atributo background-repeat: com os valores repeat, quando queremos criar um fundo quadriculado onde a imagem se repete, no-repeat, se não queremos que se repita, repeat-x para repetir na horizontal e repeat-y para repetir a imagem na vertical.

Por padrão, a posição de referência é left-top (esquerda acima) que indica que é a partir desse ponto que a imagem começará a ser repetida, mas podemos mudar essa referência para center-top (centro acima), right-top (direita acima), left-center (esquerda ao centro), center-center(centro), right-center (direita ao centro), left-bottom (esquerda abaixo), center-bottom (centro abaixo) e right-bottom(direita abaixo).

Redimensionar

Para redimensionar a imagem utilizamos a propriedade background-size com os valores auto para que a imagem seja aplicada ao centro e em seu tamanho original, length (px ou %) para redimensionar a largura da imagem de forma personalizada, cover para fazer com que seja exibida totalmente sem cortes ou contain para redimensionar dentro do container mesmo que precise cortar a imagem para poder caber.

Vínculo/ attachment

Com a propriedade attachment podemos fixar a imagem de fundo com fixed ou deixar rolando junto com o conteúdo utilizando o valor scroll.

Shorthand para o background

Também podemos usar a forma encurtada para configurar o background adicionando na ordem a cor, a imagem, a posição, a repetição e o vinculo/attachment:

  • background: black url('Imagens/g.png') center center no repeat fixed;

Icones/ Fivecons

Para adicionar um favicon em sua página existem 3 opções:

Icone na pasta local

<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

Salve o fivecom em uma pasta e dentro da tag <head> digite "link", escolha a opção "link:favicon", depois, em "href" digite "Ctrl + espaço" e localize o ícone desejado. Se ele estiver em alguma pasta anterior digite "../" para acessá-la.

Baixando um ícone pronto

Se preferir poderá baixar um ícone do site IconArchive.com pronto para usá-lo em seus projetos. Lá você pode pesquisar por temas e usar posicionando conforme mencionado anteriormente.

Criar um ícone

O mais recomendado é criar seu próprio ícone, baixá-lo e posicionar em sua página. Para isso, acesse favicon.cc e crie o seu, do seu jeito.

Trasforme um arquivo em icone

Outra opção é transformar um arquivo PNG, um texto ou um emoji em ícone usando o site favicon.io.

Outros formatos de favicons

O formato mais aceito por todos os navegadores é o ico, mas você pode usar outros como:

  • PNG
  • JPG
  • SVG
