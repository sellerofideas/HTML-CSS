Fontes em CSS

Trabalhando com fontes em Css

Para trabalhar com as fontes existem 2 formas mais usadas:

  • Usando o estilo de CSS interno

    • Nesse caso usamos a tag style dentro da head na própria página. o código fica assim: 

    
<style>
    body{
        font-family: Arial, Helvetica, sans-serif;
        font-size: 30px;
    }
    h1{
        font-family: 'Courier New', Courier, monospace;
        font-size: 50px;
    }
    p{
        font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
        font-size: 20px;
    }
</style>

  • Usando o estilo CSS externo

    • Nesse caso as configurações serão feitas em uma folha de CSS externa. Confira nosso conteúdo sobre estilos de CSS aqui.

Tamanho das fontes

Para configurar o tamanho das fontes em CSS usamos o parâmetro "font-size:".

Existem dois tipos de tamanhos de fontes:

  1. Medidas Absolutas

    2. Cm (centímetro), mm (milímetro), in (polegada), px (pixel), pt (ponto), pc (paica).

  2. Medidas Relativas.

    3. "Em" (Medida relativa ao tamanho da letra M maiúsculo da fonte), "ex" (Relativa ao tamanho da letra x da fonte), "rem" (Relativa a fonte configurada no body), "vw" (Relativa ao percentual da largura largura da Viewport ou tamanho da tela), "vh" (relativo ao percentual da altura da viewport), % (Porcentagem relativa).

    Os mais utilizados para o tamanho da fonte são o "em" e o "px".
    Normalmente 16px é o mesmo tamanho que 1em. É sempre recomendado usar em para se adequar melhor ao tamanho das telas, desde um relógio até uma TV.

Confira sobre a medida suportada das fontes no site da M3C aqui.

Peso das fontes.

Para configurar o peso usamos o comando "font-weight:". Os tipos de pesos são:

  • Lighter
  • Normal
  • Negrito (bold)
  • Extra-bold
  • Numérica de 100 á 900.

Estilo das fontes

Para usar estilo nas fontes configuramos usando "font-style:", assim podemos acessar as configurações:

  • Italico
  • Oblíquo

Para simplificar os comandos existe uma configuração chamada "shorthand" que une várias configurações em uma única linha deixando o código mais enxuto, a sequência que deve ser aplicada é font-style, font-weight, font-size, font-family.

    O código fica:
  • font: italic bolder 3em 'work sans', sans-serif

Shorthands

Podemos simplificar o uso das fontes em CSS usando as "shorthands" que são formas abreviadas de escrever um código.

O jeito normal de configurar uma fonte é:


    p{
        font-family: Arial, Helvetica, sans-serif;
        font-size: 1em;
        font-style: italic;
        font-weight: bold;
    }

Com o uso de shorthand:


    p{
        font: italic bold 1em Arial, Helvetica, sans-serif;
    }

A ordem é muito importante no caso das shorthands, no caso da propriedade fonte a ordem é:

  • font-style
  • font-variant
  • font-weight
  • font-size/line/height
  • fontfamily

Formas de Alinhamento

Para alinhar conteúdos em CSS usamos na tag head ou em uma folha de estilo separada as configurações de "text-", "text-align:", "text-indent" etc.

Existem quatro formas de alinhamento de textos:

  • text-align: left;
  • text-align: right;
  • text-align: center;
  • text-align: justify;

Para implementar uma fonte do Google fontes é preciso escolher a fonte desejada, copiar o código CSS e implementar na "head", na tag "style" ou no arquivo CSS com as configurações desejadas.

  • Truetype(ttf)
