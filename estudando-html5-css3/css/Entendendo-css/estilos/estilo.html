Usando CSS para dar estilo no conteúdo do site

CSS externo

Usando essa técnica você consegue deixar todo o site em um só padão.
Dentro da área da "head" onde são feitas as configurações da pagina digite "link" e escolha "link:css". Depois pressione "Ctrl" e clique no nome do arquivo css que foi criado para adicionar as configurações desejadas, assim todas as páginas que estiverem lincadas a esse arquivo css serão mudadas e padronizadas simultâniamente.

    
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="shortcut icon" href="../../Imagens/favicon.ico" type="image/x-icon">
        <title>Estilos
        <link rel="stylesheet" href="../../css/style1.css/estilo.css">    
    </head>
    <body>

CSS interno2

A técnica CSS interno permite mudar vários elementos que estão dentro de uma página ao mesmo tempo. Ela serve para alterar elementos como títulos, parágrafos entre outros elementos que se repetem na página sem a nescessidade de alterar 1 por 1 e caso você queira uma mudança que não afete outras páginas.
Para usar essa técnica basta adicionar as configurações CSS desejadas dentro da "head", assim a página será alterada.

    
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="shortcut icon" href="../../Imagens/favicon.ico" type="image/x-icon">
        
            <style>
                h1{
                    counter-reset: red;
                    font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
                }
                h2{
                    margin: 0px;
                    padding: 5px;
                }
            </style>
        
        <title>Estilos    
    </head>
    <body>

CSS inline

A técnica INLINE é usada caso queira alterar somente um determinado elemento da página deixando ele com a cor darkcyan, por exemplo. Para isso, basta adicionar o código CSS na tag da linha desejada.

<p style="color: brown;">Técnica CSS inline</p>
